Flare3DLoader1()Constructor
public function Flare3DLoader1(request:*, parent:Pivot3D = null, sceneContext:Scene3D = null)

Creates a new Flare3DLoader loader.

Parameters
request:* — Path of the file to be loaded, ByteArray or Class.
 
parent:Pivot3D (default = null) — When a F3D file is loaded, this may contain multiple 3d objects. The Flare3DLoader object can be the container for all these objects or another parent can be specified (either a scene or another Pivot3D to contain them). The default value is ‘null’. If another parent is specified, the Flare3DLoader object will remain empty.
 
sceneContext:Scene3D (default = null)
